 <h1>Sistema de Autenticação</h1>

  <p>
    Projeto simples de estudos com login e cadastro de usuários.
    Criado para aprendizado e uso em portfólio.
  </p>

  <p>
    O backend foi desenvolvido com auxílio de Inteligência Artificial
    para fins educacionais.  
    O frontend (design) foi desenvolvido por mim utilizando Tailwind CSS.
  </p>

  <h2>Funcionalidades</h2>
  <ul>
    <li>Cadastro de usuários</li>
    <li>Login</li>
    <li>Senhas criptografadas</li>
    <li>Conexão com banco de dados MySQL</li>
  </ul>

  <h2>Tecnologias</h2>
  <ul>
    <li>Node.js</li>
    <li>Express.js</li>
    <li>MySQL</li>
    <li>JavaScript</li>
    <li>Tailwind CSS</li>
  </ul>
